# Vibe Library

A visual prompt library built with Next.js.

Browse curated prompt cards across categories like Prompt Engineering, Coding, Problem Solving, Learning, Image, UI/UX, and more. The app also includes a Prompt Crafter, full-text prompt search, and an animated landing page with a reactive CTA.

## Stack

- Next.js 16
- React 19
- TypeScript
- CSS Modules
- GSAP
- Three.js / React Three Fiber

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
